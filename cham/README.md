## **개요**

간단하게 편한 쇼핑몰

### Scripts

실행시키기 위해 FrontEnd, BackEnd 둘다 package를 다운받고 실행시켜야 한다.

#### `npm install`

FrontEnd,BackEnd 폴더에서 npm install

#### `npm start`

FrontEnd,BackEnd 폴더에서 npm start하면 http://localhost:3000에서 실행된다.
